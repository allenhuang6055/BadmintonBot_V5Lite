# 健好羽球 LINE Bot V9 正式升級版

## 本版重點

- 收入與單純耗球自動分開顯示。
- 庫存負數顯示修正，不再出現 `-4桶 + -9顆`。
- 私訊記帳成功後自動推送群組。
- 如果在群組直接記帳，Bot 只回覆群組，不重複推送。
- 新增 `群組通知設定`。
- 新增 `群組測試`。
- 月報加入目前資金餘額、安全水位、財務狀態、庫存狀態。
- 保留 V8.2 的 `今天`、`庫存`、`月報`、每日報表。

# V9.1 更新：收入解析容錯

## 支援更寬鬆輸入

會員：
`會員800`、`會員 800`、`會員：800`、`會員費800`、`會費 800`、`年費800`、`會員 800元`、`會員：８００`

球券：
`球券600`、`球卷600`、`球資600`、`球費600`

零打：
`零打500`、`臨打500`、`零打收入500`

耗球：
`耗球10`、`用球10`、`球耗10`

## Render Logs

每次收入解析會出現：

```text
PARSE_INCOME_RESULT
```

可直接看到每個項目解析到多少。

# V9.2 更新：模糊辨識防漏

新增：
- `球劵` 會正確辨識為 `球券`
- 模糊辨識常見錯字
- 未辨識項目不再默默漏掉，Bot 會在回覆中提醒
- Render Logs 的 `PARSE_INCOME_RESULT` 會列出 matched / unknown / result

範例：

```text
零打：1200元
球劵：1600元
贊助：1000元
耗球：20顆
```

會寫入：

```text
零打 1200
球券 1600
贊助 1000
耗球 20
```
